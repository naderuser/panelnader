## 🎯 معرفی

پنل ارزشیابی توصیفی یک ابزار آنلاین برای معلمان دوره ابتدایی است که امکان تولید گزارش توصیف عملکرد دانش‌آموزان را فراهم می‌کند.

---

## ✨ ویژگی‌ها

### 📊 تولید گزارش توصیف عملکرد
- **پایه‌های تحصیلی**: اول تا ششم ابتدایی
- **دروس**: فارسی، ریاضی، علوم، قرآن، اجتماعی
- **بازخوردها**: 
  - ⭐ خیلی خوب
  - 👍 خوب
  - 📝 قابل قبول
  - 💪 نیاز به تلاش بیشتر
- **نوبت تحصیلی**: نوبت اول و نوبت دوم
- **قابلیت کپی**: کپی تکی و کپی همه توصیف‌ها

### 🤖 هوش مصنوعی Groq
- چت متنی با هوش مصنوعی
- **فقط متن** (بدون عکس و فایل)
- کمک در نوشتن توصیف عملکرد
- API Key قابل تنظیم در پنل

---

## 🚀 نحوه استفاده

### ۱. دیپلوی در Cloudflare Workers

1. وارد سایت [Cloudflare Dashboard](https://dash.cloudflare.com) شوید
2. یک Worker جدید بسازید
3. کد `worker.js` را کپی کنید
4. یک **KV Namespace** با نام `EXAM_KV` بسازید
5. آن را به Worker متصل کنید

### ۲. تنظیمات

#### API Key هوش مصنوعی
1. وارد پنل معلم شوید (`/teacher`)
2. به بخش **تنظیمات** بروید
3. API Key Groq را وارد کنید
   - از [console.groq.com](https://console.groq.com) دریافت کنید

### ۳. استفاده از تولید عملکرد

1. نام دانش‌آموز را وارد کنید
2. کلاس را مشخص کنید
3. نوبت تحصیلی را انتخاب کنید
4. پایه تحصیلی را انتخاب کنید
5. توصیف‌های عملکرد را مشاهده کنید
6. با دکمه **کپی** توصیف‌ها را کپی کنید
