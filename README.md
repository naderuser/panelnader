# 🏫 پنل آموزشی هوشمند - Cloudflare Worker

یک پنل آموزشی کامل و حرفه‌ای طراحی شده برای اجرا روی **Cloudflare Workers** با **Cloudflare KV** به عنوان دیتابیس.

## ✨ قابلیت‌ها

### 👨‍🏫 پنل معلم
- **مدیریت کلاس‌ها** - ایجاد، ویرایش و حذف کلاس‌ها
- **مدیریت دانش‌آموزان** - ثبت نام، رمز اختصاصی برای هر دانش‌آموز
- **آزمون‌ساز** - سوالات تشریحی، چهارگزینه‌ای، صحیح/غلط، کوتاه‌پاسخ
- **تصحیح و بازخورد** - نمره‌دهی و ارسال بازخورد به دانش‌آموزان
- **حضور و غیاب** - ثبت حضور دانش‌آموزان به تفکیک کلاس و تاریخ
- **نمرات** - ثبت نمرات دروس مختلف
- **تقویم آموزشی** - مناسبت‌ها، تکالیف و امتحانات
- **خروجی Word** - دانلود برگه آزمون و پاسخنامه

### 🎥 کلاس درس آنلاین (مهم‌ترین قابلیت)
- معلم **لینک کلاس** می‌سازد
- معلم **فایل آپلود می‌کند** (ویدیو، عکس، PDF، Word)
- دانش‌آموزان با حساب کاربری وارد می‌شوند
- **فقط دانش‌آموزان می‌توانند چت کنند**
- معلم می‌تواند پیام‌ها را مشاهده کند

### 👨‍🎓 پنل دانش‌آموز
- **کلاس‌های من** - مشاهده کلاس‌های ثبت‌نام شده
- **منابع آموزشی** - دانلود فایل‌های آپلود شده توسط معلم
- **چت کلاس** - ارسال پیام در کلاس درس
- **کارنامه** - مشاهده نمرات
- **تقویم** - رویدادها و تکالیف

### 🛠️ ابزارهای کمکی
- **اسکنر عکس** - بهبود کیفیت با فیلترهای سیاه/سفید، خاکستری، سند
- **کاهش حجم** - فشرده‌سازی عکس‌ها با تنظیم کیفیت
- **جدول‌ساز** - ساخت جدول و خروجی Excel

### 🤖 هوش مصنوعی
- چت‌بات با قابلیت پاسخ به سوالات آموزشی (نیاز به GROQ_API_KEY)

## 📦 نصب و راه‌اندازی

### ۱. پیش‌نیازها
- Node.js 18+
- حساب Cloudflare

### ۲. ساخت KV Namespace
```bash
# در داشبورد Cloudflare:
# Workers & Pages > KV > Create a namespace
# نام: EDUCATIONAL_DB
```

### ۳. تنظیم wrangler.toml
```toml
[[kv_namespaces]]
binding = "DB"
id = "YOUR_KV_NAMESPACE_ID"  # ← آیدی کپی شده از Cloudflare
```

### ۴. دیپلوی
```bash
npm install -g wrangler
wrangler login
wrangler deploy
```

### ۵. تنظیم متغیرهای محیطی (اختیاری)
در Cloudflare Dashboard > Workers > your-worker > Settings > Variables:
- `GROQ_API_KEY`: کلید API از [console.groq.com](https://console.groq.com)

## 🚀 استفاده

### پنل معلم
به آدرس `/teacher` بروید. در اولین ورود، رمز عبور تعیین کنید.

### پنل دانش‌آموز
به آدرس `/student` بروید. با کد دانش‌آموزی و رمزی که معلم داده وارد شوید.

### کلاس درس
1. معلم یک "کلاس درس" می‌سازد
2. دانش‌آموزان را به کلاس اضافه می‌کند
3. فایل آپلود می‌کند
4. دانش‌آموزان از پنل خود وارد کلاس می‌شوند

## 🔧 API Endpoints

### معلم
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/teacher/login | ورود |
| POST | /api/teacher/logout | خروج |
| GET | /api/teacher/classes | لیست کلاس‌ها |
| POST | /api/teacher/classes | ایجاد کلاس |
| GET | /api/teacher/students | لیست دانش‌آموزان |
| POST | /api/teacher/students | ثبت دانش‌آموز |
| POST | /api/teacher/classrooms | ایجاد کلاس درس |
| POST | /api/teacher/classrooms/:id/resources | آپلود فایل |
| POST | /api/teacher/attendance | ثبت حضور و غیاب |
| POST | /api/teacher/grades | ثبت نمره |

### دانش‌آموز
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/student/login | ورود |
| GET | /api/student/classrooms | کلاس‌های من |
| GET | /api/student/classrooms/:id/resources | منابع |
| POST | /api/student/classrooms/:id/messages | ارسال پیام |
| GET | /api/student/grades | نمرات |

## 📝 نکات مهم

- تمام داده‌ها در **Cloudflare KV** ذخیره می‌شوند (رایگان تا 1GB)
- حداکثر حجم فایل آپلود: **5MB**
- فایل‌ها به صورت **Base64** ذخیره می‌شوند
- احراز هویت با **HttpOnly Cookies** و **SHA-256 Hash**
