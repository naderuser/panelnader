# 📚 پنل آموزشی جامع

یک Cloudflare Worker کامل برای مدیریت آموزشی

## ✨ امکانات

### 🔐 پنل معلم
- ورود/خروج با رمز عبور امن (SHA-256)
- تغییر رمز عبور
- تم روشن/تاریک

### 👨‍🎓 مدیریت دانش‌آموزان
- افزودن/حذف دانش‌آموز
- لینک اختصاصی برای هر دانش‌آموز
- سوال امنیتی برای ورود

### 📝 آزمون‌ساز
- **سوالات تشریحی** - پاسخ متنی آزاد
- **سوالات چهارگزینه‌ای** - تصحیح خودکار
- **صحیح/غلط** - پاسخ بله/خیر
- **کوتاه‌پاسخ** - پاسخ کوتاه

### 🛠️ ویرایشگر غنی سوالات
- علائم ریاضی (×÷−+√∑)
- کسر و تقسیم چکشی
- اشکال هندسی SVG
- آپلود عکس

### ✅ تصحیح و بازخورد
- تصحیح دستی برای تشریحی
- تصحیح خودکار برای چندگزینه‌ای
- پاسخنامه با وضعیت‌های مختلف

### 📅 برنامه هفتگی
- ساخت جدول ۵ روزه با ۵ زنگ
- خروجی Word/PDF/چاپ
- ذخیره در Cloudflare KV

### 📊 جدول‌ساز حرفه‌ای
- خروجی اکسل RTL
- میانگین‌گیری خودکار
- رنگ‌بندی سلول‌ها

### 📷 ابزارهای تصویری
- **اسکنر حرفه‌ای** - مشابه CamScanner
- **کاهش حجم عکس** - با کیفیت و فرمت‌های مختلف
- **برش عکس** - با نسبت‌های مختلف
- **تبدیل PDF به عکس** - با انتخاب صفحات و DPI

### 🤖 چت AI و ترجمه
- چت AI با Groq (چهار حالت)
- ترجمه متن با MyMemory

## 🚀 نصب

### مراحل:

1. **ساخت Worker در Cloudflare**
   - به [Cloudflare Dashboard](https://dash.cloudflare.com) بروید
   - یک Worker جدید بسازید
   - کد `worker.js` را کپی کنید

2. **ساخت KV Namespace**
   - در Cloudflare Dashboard به Workers & Pages بروید
   - یک KV namespace با نام `EXAM_KV` بسازید

3. **اتصال KV به Worker**
   - در تنظیمات Worker، KV binding اضافه کنید
   - نام: `EXAM_KV`

4. **Deploy**
   - Worker را deploy کنید

## 🔧 متغیرهای محیطی

| متغیر | توضیحات | ضروری |
|--------|---------|--------|
