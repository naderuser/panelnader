# 📚 پنل آموزشی جامع

نسخه اندروید اپلیکیشن پنل آموزشی - ساخته شده با Flutter

![Designer](https://img.shields.io/badge/Designer-Nader%20Akseki-purple)
![Platform](https://img.shields.io/badge/Platform-Android-green)
![Flutter](https://img.shields.io/badge/Flutter-3.24.0-blue)

## ✨ امکانات

### 📅 برنامه هفتگی
- ساخت جدول ۵ روزه با ۵ زنگ
- خروجی Word/PDF/چاپ
- ذخیره در سرور

### 📊 جدول‌ساز حرفه‌ای
- خروجی اکسل
- میانگین‌گیری خودکار
- رنگ‌بندی سلول‌ها

### 📷 ابزارهای تصویری
- **اسکنر حرفه‌ای** - مشابه CamScanner با فیلترهای متنوع
- **کاهش حجم عکس** - با کیفیت و فرمت‌های مختلف
- **برش عکس** - با نسبت‌های مختلف
- **تبدیل PDF به عکس** - با انتخاب صفحات و DPI

### 🤖 چت AI
- چت هوشمند با Groq API
- پاسخگویی به سوالات آموزشی

### 📋 طرح درس
- ساخت طرح درس روزانه
- خروجی PDF و چاپ

### ⚙️ تنظیمات
- تم روشن/تاریک
- تنظیمات API

## 🚀 نصب

### روش اول: نصب APK
فایل `panel_amuzeshi.apk` را از بخش Releases دانلود کرده و روی گوشی اندرویدی نصب کنید.

### روش دوم: ساخت از سورس
```bash
flutter pub get
flutter build apk --release
```
